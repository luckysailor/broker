# QuickbaseSync

Automate importing and exporting of data from Quickbase applications using post requests.

## Installation

Add this line to your application's Gemfile:

```ruby
    gem quickbase_sync
```

And then execute:

```ruby
    $ bundle
```

Or install it yourself as:

```ruby
    $ gem install quickbase_sync
```
