# QuickbaseSync

Automate importing and exporting of data from Quickbase applications using post requests.

## Installation

Add this line to your application's Gemfile:

    gem 'quickbase_sync'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install quickbase_sync
